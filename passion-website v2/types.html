Pokémon come in a couple different types. A type is a category that defines the element their moves utilize, and is used to measure how effective those skills are against other Pokémon. For example, a water type Pokémon would have moves such as Water Gun, or Whirlpool, while a fire type Pokémon would have moves like Flamethrower or Fire Spin. Pokémon can also have up to two types, makig them dual-type Pokémon, but moves can only be one type. Most if not all dual-type Pokémon's types correspond with each other. Some Pokémon have dual types with two types that don't go together, such as water/electric or grass.flying types. Pokémon can also gain a different type upon evolution. Overall, there are now 18 total official types of Pokémon.

  • Normal
  • Fire
  • Water
  • Grass
  • Electric
  • Ice
  • Fighting
  • Poison
  • Ground
  • Flying
  • Psychic
  • Bug
  • Rock
  • Ghost
  • Dragon
  • Dark
  • Steel
  • Fairy

As mentioned previously, each type has its own strengths and weaknesses, most of which follow a logical setup. For example, fire type Pokémon are weak against water type Pokémon, ice type Pokémon are weak against fire type Pokémon, water type Pokémon are weak against electirc type Pokémon, and so forth.
